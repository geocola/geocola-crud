addFilter

  • function
 

Adds a new filter or set of filters to the list of filters in this widget. A filterFactory may be defined on the field which may return one filter or an array of filters.

function(scope, dom, event, obj)

Parameters

  1. scope {can.Map}

    The stache scope

  2. dom {event}

    The dom event

  3. event {event}

    The can event

  4. obj {filterObject}

    The object to add. This is the only argument used by the function, the rest may be null.
