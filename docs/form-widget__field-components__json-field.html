<json-field />

  • module
form-widget/field-components/json-field

{can.Component}

 

This component creates a sub form that is serialized to a JSON string when values in the sub form change. This can be used to store JSON data in a database field.

can.Component

Properties

  1. type {String}

    This should be set to json for the form widget to use this template

  2. properties.template {can.Map}

    The object to use as the form object template

  3. properties.formFields {Array<String | FormFieldObject>}

    The form fields to use in the form
