deleteObject

  • function
 

Displays a confirm dialog box and if confirmed, deletes the object provided. Once the object is deleted, a message is published using PubSub.

deleteObject(obj, skipConfirm)

Parameters

  1. obj {can.Map}

    The object to delete

  2. skipConfirm {Boolean}

    If true, the method will not display a confirm dialog

deleteObject( scope, dom, event, obj, skipConfirm )

Parameters

  1. scope {can.Map}

    The stache scope (not used)

  2. dom {domNode}

    The domNode that triggered the event (not used)

  3. event {Event}

    The event that was triggered (not used)

  4. obj {can.Map}

    The object to delete

  5. skipConfirm {Boolean}

    If true, the method will not display a confirm dialog and will immediately attempt to remove the object
